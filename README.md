# BackMEAN
Back end side of the 2021 web programming class MEAN stack project with Pr. Michel Buffa

Front end available [here](https://github.com/nsdev74/FrontMEAN)

## Overview

![Demonstration](https://s3.gifyu.com/images/ezgif.com-gif-to-mp4.gif)

### Setup

Clone the repository using the following command on your git bash terminal:

```
$ git clone https://github.com/nsdev74/BackMEAN.git
```


Navigate to the project's directory and run this command to install the dependences:

```
npm install
```

Run the Back end server using the following:

```
npm start
```

The database is already running on the cloud @ MangoDB and does not need to be configured.
